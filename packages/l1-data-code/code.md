## Задание

Нужно закодировать 33 буквы кириллицы, 10 цифр (0 - 9) и знаки препинания

- Точка (.)
- Запятая (,)
- Точка с запятой (;)
- Двоеточие (:)
- Многоточие (...)
- Вопросительный знак (?)
- Восклицательный знак (!)
- Тире (—)
- Скобки (( ), [ ]) - 4 знака (открывающая и закрывающая)
- Кавычки (« », „ “) - 4 знака (открывающая и закрывающая)
- Пробел (\n)
- Табуляция (\t)
- **Итого**: 18 знаков

Всего знаков необходимо закодировать *33 + 10 + 18 = 61*.

Если верить [Графику частотности использования букв](https://pikabu.ru/story/grafik_chastotyi_bukv_v_russkom_yazyike_7628779), то в длинном тексте 50% символов это О, Е, А, И, Н, С, Т.

В таком случае предлагается для самых частых символов использовать самые короткие последовательности битов.

### Алгоритм

Первые два бита управляющие, затем идут биты данных.
Биты данных будут составлять минимальный 2 бит, максимальный 6 бит. В таком случае можно будет закодировать
*4 + 8 + 64* = 76 символов


| Бит  | Число | Действие          |
| ---- | ----- | ----------------- |
| `00` | `0`   | CAPS              |
| `01` | `1`   | Прочитать 2 бита  |
| `10` | `2`   | Прочитать 3 бита  |
| `11` | `3`   | Прочитать 6 битов |

В таком случае наиболее частотные символы в сумме будут занимать 4 бита, наименее частотные, знаки препинания и цифры 8 бит.

#### Примеры

Формула подсчёта общего количества бит:  
`COUNT = 2 * N_CAPS + 4 * N_2bit + 5 * N_3bit + 8 * N_6bit`, где:

- `N_CAPS` — число переключений регистра (`00 (CAPS)`)
- `N_2bit` — число символов из таблицы 2-битных данных (`01 + 2 бита`)
- `N_3bit` — число символов из таблицы 3-битных данных (`10 + 3 бита`)
- `N_6bit` — число символов из таблицы 6-битных данных (`11 + 6 бит`)
